### Pip ###

#### Install pip ####
    1. download get-pip.py
    2. python get-pip.py

    on debian and ubuntu:
        **install pip for python2** 
        sudo apt-get install python-pip

        **install pip for python3** 
        sudo apt-get install python3-pip
    

#### Update pip ####
    1. pip install -U pip
